Minimum jumps required to reach the end of the array. 
The program find the minimum jumps that you need to reach the end of the array. The value inside the array indicates the number of jumps that you can take. If you are stuck at an array where the value is 0, then that means that you cannot reach the array. 

language: 
MinimmJump = {< array , array_values> | The array[0] > 0}
